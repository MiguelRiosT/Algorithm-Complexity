Thermostat      

Alguien reconfiguró el antiguo termostato a una temperatura de "a"

El termostato solo puede ser ajustado a una temperatura 
de l a r
La temperatura no puede cambiar a menos de x

Formalmente, en una sola operación se puede reconfigurar el termostato desde temperatura a
 a la temperatura b solamente si esto sucede
 si  |a−b|≥x and l≤b≤r

Te dan 
-l
-r 
-x
-a y b

Encuentre el número mínimo de operaciones 
requeridas para obtener la temperatura b de la temperatura a
o decir que es imposible.


//////////////////////////INPUT!//////////////////////////
1.La primera línea de datos de entrada contiene el único entero t
    dicho t presenta 
    (1≤t≤10^4) --> el número de casos de prueba en la prueba.

TEST CASES 
1.La primera línea de cada caso contiene tres números enteros l,r y x
(−10^9≤l≤r≤10^9, 1≤x≤10^9) --> rango de temperatura y cambio mínimo de temperatura

2.La segunda línea de cada caso contiene dos enteros a y b
(l≤a,b≤r) -->  las temperaturas inicial y final.

//////////////////////////OUTPUT!//////////////////////////
Salida t números, cada uno de los cuales es la respuesta al caso de prueba correspondiente. Si es imposible alcanzar la temperatura b
,mostrar -1, de lo contrario, mostrar el número mínimo de operaciones.
